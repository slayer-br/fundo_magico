# Fundo Mágico

## Descrição

O Fundo Mágico é uma aplicação web que permite aos usuários gerar backgrounds personalizados para seus sites ou projetos, utilizando o poder da Inteligência Artificial. Basta descrever o fundo que você imagina, e a aplicação gera o código HTML e CSS correspondente, além de uma visualização em tempo real.

Este projeto foi desenvolvido durante a "Semana do Zero ao Programador Contratado" (29/09/2025 a 05/10/2025), um evento focado em ensinar programação e desenvolvimento de software.

## Funcionalidades

- Geração de backgrounds a partir de descrições em texto.
- Visualização do resultado em tempo real.
- Exibição dos códigos HTML e CSS gerados para fácil utilização.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- API de Inteligência Artificial para geração de código.
- n8n para automação de workflows.

## Como Utilizar

1. Abra o arquivo `index.html` em seu navegador.
2. No campo de texto, descreva o background que você deseja criar. (Ex: "gradiente de azul claro para azul escuro")
3. Clique no botão "Gerar Background Mágico".
4. Visualize o resultado e copie os códigos HTML e CSS para utilizar em seus projetos.
